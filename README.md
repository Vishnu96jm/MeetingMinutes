# MeetingMinutes

This is a simple app to track/ log minutes of meetings(MOM).

**Features:**
* Note taking app to write notes and to-do lists
* Signing-in using Email and password
* Add / Edit a MOM note

**Architecture components and libraries used:**
* **MVVM** architecture pattern is used with **ViewModel, LiveData**.
* Uses Firebase authentication for sign-in / sign-up, Firestore for backend support.
* Uses **View Binding** to generate a binding class for each XML layout file present in that module.
